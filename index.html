What I Do

  • HTML

    HTML은 다양한 요소와 태그를 사용하여 텍스트, 이미지, 링크, 비디오 등 다양한 콘텐츠를 정의하며, 웹 브라우저가 이를 해석하여 사용자에게 시각적으로 표현합니다. HTML은 웹의 기본 골격을 형성하며, CSS(스타일시트)와 JavaScript(스크립트 언어)와 함께 사용되어 웹 페이지의 디자인과 기능을 확장합니다.

  • CSS

    CSS는 웹 페이지의 시각적 스타일과 레이아웃을 정의하는 스타일 시트 언어입니다. CSS는 HTML 요소의 색상, 글꼴, 여백, 배치 등을 지정하여 웹 페이지의 디자인을 향상시키며, 다양한 화면 크기에 맞춰 반응형 디자인을 구현하는 데에도 사용됩니다.

  • C

    C는 1970년대 초 데니스 리치에 의해 개발된 고급 프로그래밍 언어로, 시스템 프로그래밍과 응용 프로그램 개발에 널리 사용됩니다. 효율성과 성능이 뛰어나며, 하드웨어에 가까운 저수준 프로그래밍과 메모리 관리 기능을 지원합니다. C는 UNIX와 여러 현대 언어의 기반이 되는 중요한 언어입니다.
